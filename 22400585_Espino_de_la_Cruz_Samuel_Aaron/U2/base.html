Que es GitHub

GitHub es una plataforma en la nube que permite almacenar, compartir y colaborar en proyectos de código.

Utiliza como base Git, un sistema de control de versiones que registra y administra cambios en archivos, facilitando el trabajo en equipo sin sobrescribir el trabajo de otros.
