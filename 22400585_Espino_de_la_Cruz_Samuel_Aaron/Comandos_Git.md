Comandos Git
1. git init

Descripción: Inicializa un repositorio Git en una carpeta.
Ejemplo de uso: git init

2. git clone

Descripción: Clona un repositorio remoto a la computadora local.
Ejemplo de uso: git clone https://github.com/usuario/proyecto.git

3. git status

Descripción: Muestra el estado actual de los archivos del repositorio.
Ejemplo de uso: git status

4. git add

Descripción: Agrega un archivo al área de preparación (staging).
Ejemplo de uso: git add archivo.txt

5. git add .

Descripción: Agrega todos los archivos modificados al área de preparación.
Ejemplo de uso: git add .

6. git commit

Descripción: Guarda los cambios preparados en el historial del repositorio.
Ejemplo de uso: git commit -m "Corrección de errores"

7. git log

Descripción: Muestra el historial de commits realizados.
Ejemplo de uso: git log

8. git diff

Descripción: Muestra las diferencias entre archivos o versiones.
Ejemplo de uso: git diff

9. git branch

Descripción: Permite listar o crear ramas.
Ejemplo de uso: git branch desarrollo

10. git checkout

Descripción: Cambia a una rama específica.
Ejemplo de uso: git checkout desarrollo

11. git switch

Descripción: Cambia entre ramas utilizando la sintaxis moderna de Git.
Ejemplo de uso: git switch desarrollo

12. git merge

Descripción: Fusiona una rama con la rama actual.
Ejemplo de uso: git merge desarrollo

13. git pull

Descripción: Descarga y fusiona cambios desde un repositorio remoto.
Ejemplo de uso: git pull origin main

14. git push

Descripción: Envía los cambios locales al repositorio remoto.
Ejemplo de uso: git push origin main

15. git fetch

Descripción: Descarga los cambios del repositorio remoto sin fusionarlos.
Ejemplo de uso: git fetch origin

16. git remote -v

Descripción: Muestra los repositorios remotos configurados.
Ejemplo de uso: git remote -v

17. git rm

Descripción: Elimina archivos del repositorio y del sistema de archivos.
Ejemplo de uso: git rm archivo.txt

18. git mv

Descripción: Renombra o mueve archivos dentro del repositorio.
Ejemplo de uso: git mv documento.txt informe.txt

19. git reset

Descripción: Deshace cambios en el área de preparación o regresa a un commit anterior.
Ejemplo de uso: git reset archivo.txt

20. git stash

Descripción: Guarda temporalmente cambios sin realizar un commit.
Ejemplo de uso: git stash
